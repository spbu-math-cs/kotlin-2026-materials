## Описание

Короткое описание домашнего задания, ключевые темы и ожидаемые результаты.

## Как работать с репозиторием

1. Сделайте форк этого репозитория.
2. Решайте задачи по порядку: `task-01`, `task-02`, ...
3. Реализуйте недостающий код в местах, отмеченных `TODO()`.
4. Запускайте тесты локально.

## Как запускать тесты

Из корня репозитория:

```bash
./gradlew test
```

Запуск тестов для конкретной задачи:

```bash
./gradlew :task-01:test
```

## Задачи

> **Задача** — это отдельный Gradle-модуль (`task-01`, `task-02`, ...).
> В каждой задаче может быть несколько **подзадач**.
> Задача считается выполненной тогда и только тогда, когда выполнены **все подзадачи**.
